# Chart

Read this file first when the task involves creating or reconfiguring a chart block, working with chart `query / visual / events`, doing chart readback, or maintaining chart-contract regressions.

## Routing

- By default, read [chart-core.md](./chart-core.md) first to narrow runtime setup, reconfiguration, and readback.
- If this chart configuration involves `visual.raw` or `events.raw`, also read [js.md](./js.md) and [runjs-runtime.md](./runjs-runtime.md).
- Only continue into [chart-validation.md](./chart-validation.md) when you are maintaining contract cases, negative-case matrices, or regression cases.
