# Plugin Eval Benchmark Workspace

This workspace is intentionally minimal.

- Use the installed `nocobase-ui-builder` skill as the source of truth.
- Write benchmark artifacts only under `.artifacts/nocobase-ui-builder/<scenario-id>/`.
- The fixed benchmark scenario ids are `whole-page-blueprint`, `localized-reaction-edit`, and `boundary-handoff`.
- Do not expect a live NocoBase server here.
- Post-run verification uses `scripts/verify-plugin-eval-artifacts.mjs`.
